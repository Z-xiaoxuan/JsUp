# == VS ===
== 相等，数据类型不同，先转换为一样的再比较

=== 绝对相等，必须类型和值都相同才为True（switch判断中，每一种case值的比较都是基于===来完成的）

# i++/i--
```js
let i = '10'
i = i + 1 // '101'
i += 1 //'101'
i++ // 11 他是纯粹的数学运算
```
```js
let i = 1
5+(i++) //先算5+i=6，然后 i 再加1

let i = 1
5+(++i) //先i累加1，然后那累加后的结果去+5 = 7

//题目
let i = 3
console.log(5 + (++i) + (i++) + 3 - 2 + (--i) + (i--) -2)
//运算正确 得数20 i为3
```
